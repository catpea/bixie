# bixie 😵‍💫

Example Usage

Create webpage in ```~/Uploads/geocities/Area51/Dreamworld/majestic12``` based on jpg,png,gif,webp and webm,mp4 in ```~/Downloads/skynet```


```bash

bixie --dist ~/Uploads/geocities/Area51/Dreamworld/majestic12 ~/Downloads/skynet

```
